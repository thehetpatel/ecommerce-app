# E-Commerce Redux Demo

A simple e-commerce shopping cart application built with React, Redux Toolkit, and Vite.
Uses [DummyJSON](https://dummyjson.com/) for mock data.

## Features
- User Login (Redux Auth)
- Product Browsing & Filtering
- Shopping Cart (Add/Remove items)
- Mock Checkout

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run development server:
   ```bash
   npm run dev
   ```
